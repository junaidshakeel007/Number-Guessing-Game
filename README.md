# Number Guessing Game

A web mini game where players guess a random number from 1 to 100 within 7 attempts.

## Live Link
https://github.com/junaidshakeel007/Number-Guessing-Game/

## Features
* Built with pure HTML CSS and JavaScript
* Players can press Enter to submit a guess
* The input box clears and focuses automatically
* Includes options to quit mid game or restart after playing
* Clean responsive layout for different screens

## How To Play
1. Click Play Game to start
2. Enter a number between 1 and 100
3. Click Check or press Enter
4. Follow the on screen hints to find the correct number
5. Win by guessing the number within 7 tries
